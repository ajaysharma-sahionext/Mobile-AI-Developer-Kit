# AI Project Standard (.ai)

This directory contains reusable AI-readable project context.

## Purpose

The `.ai` directory provides structured project knowledge that can be consumed by local or cloud AI assistants.

## Files

- project.template.md
- architecture.template.md
- rules.template.md
- stack.template.md
- features.template.md
- tasks.template.md
- decisions.template.md
- progress.template.md
- glossary.template.md
- known-issues.template.md

## Usage

1. Copy each template.
2. Rename by removing `.template`.
3. Fill in project-specific information.
4. Keep the files updated alongside the codebase.

Do not store passwords, API keys, tokens or other secrets in this directory.
