# Overview

Mobile AI Developer Kit 2026 is a complete open-source guide that explains how to convert an Android phone into a powerful AI-assisted development environment using free and open-source tools.

The goal is **not** to replace Cursor or VS Code Desktop completely.

The goal is to provide the closest possible Cursor-like development experience on Android using local AI models, Termux, VS Code, Ollama and modern developer tools.

This guide is written for:

* React Developers
* React Native Developers
* Node.js Developers
* Python Developers
* Full Stack Developers
* Students
* Open Source Contributors

---

# Target Device

This guide is primarily optimized for:

* Android 15+
* Android 16
* Snapdragon 8 Gen Series
* 8GB RAM
* 128GB Storage (Minimum)
* 25GB+ Free Storage

A separate optimization section will be included for 12GB and 16GB devices.
